# 红中赖子麻将案例训练

手机优先的《红中赖子麻将高期望收益攻略》互动学习站。

当前版本包含九章互动教学手册、32个图形化案例和一页牌桌速查。其中规则结论标记为“规则确定”，完整固定牌面模拟区间不跨0的案例标记为“1000世界支持”或“4000世界支持”，自然牌局固定核心复验标记为“固定核心1000世界”，其余继续标注为“经验性初版”。

v1.0新增：

- `textbook.html`按九章呈现理论、麻将牌例、课堂小测和反转条件。
- 章节完成状态与答题记录只保存在当前浏览器本机。
- 每章可直接跳转到对应的案例分类。
- 手册、案例与速查统一导航，并分别支持离线读取。
- 案例训练从20题扩充到32题，新增四五对碰牌、换三张、红中弃留、听牌价值与防守尾部风险题。
- 新增“红中”案例分类，第六章可直接跳转到对应训练。
- 保留原20题索引，已有浏览器学习进度不会因扩题错位。
