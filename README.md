# 红中赖子麻将案例训练

手机优先的《红中赖子麻将高期望收益攻略》案例训练站。

当前版本包含18个案例，用于验证 iPhone 阅读、答题和解析体验。其中规则结论标记为“规则确定”，完整固定牌面1000世界区间不跨0的案例标记为“千世界支持”，自然牌局固定核心复验标记为“固定核心1000世界”，其余继续标注为“经验性初版”。案例14和15采用同手牌早晚盘对照，展示七对过碰反转；案例17和18使用同一补杠手牌，展示固定必抢与当前无人能抢两个端点。

v0.7新增：

- 按换三张、开局中盘、七小对、碰牌、杠牌、听牌、防守和规则筛选。
- 选择答案后查看解析，会显示正确与错误选项。
- 已练题数和答对题数保存在当前浏览器本机，不上传任何数据。
- 页面继续支持添加到 iPhone 主屏幕和离线读取。
