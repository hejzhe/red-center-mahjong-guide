# 红中赖子麻将案例训练

手机优先的《红中赖子麻将高期望收益攻略》互动学习站。

当前版本包含九章互动教学手册、20个图形化案例和一页牌桌速查。其中规则结论标记为“规则确定”，完整固定牌面模拟区间不跨0的案例标记为“千世界支持”或“4000世界支持”，自然牌局固定核心复验标记为“固定核心1000世界”，其余继续标注为“经验性初版”。

- 每章可直接跳转到对应的案例分类。
- 手册、案例与速查统一导航，并分别支持离线读取。
- 案例页继续保留20题、分类筛选、答案解析和本机进度。
